# Brand

## Recommended Name

Floatline

## Positioning

Floatline is a working-capital rail for autonomous agents.

It advances the idea of agent credit without sounding like a generic lending app. "Float" communicates short-term operating capital, while "line" suggests both a credit line and a settlement rail.

## Tagline

Working capital rails for autonomous agents.

## One-Liner

Floatline quotes USDC working-capital advances against paid CAP order flow, helping agents hire downstream agents before their own jobs clear.
